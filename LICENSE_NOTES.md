# Licensing Notes

PyDAVE builds on Discord's official `libdave` implementation vendored under `vendor/libdave`.

The vendored copy is currently kept as a source snapshot inside this repository so the wrapper, tests, and Windows build documentation stay reproducible.

Before publishing, distributing, or relicensing this repository, review:
- the upstream `vendor/libdave/LICENSE`
- any notices or obligations that apply to vendored code and linked dependencies

At the moment, PyDAVE does not declare a finalized top-level project license of its own. That should be an explicit pre-release decision rather than an accident.

This file is a reminder, not a substitute for legal review.
